🛒 ClickKart

A full-stack e-commerce web application built using Node.js, Express.js, MongoDB, and EJS. ClickKart provides a seamless online shopping experience with secure user authentication, product management, image uploads, and shopping cart functionality.

🚀 Features
User Features
User Registration & Login
Secure Authentication using JWT
Password Encryption with Bcrypt
Browse Products
Add Products to Cart
Persistent Login using Cookies
Flash Messages for Better User Experience
Admin Features
Create New Products
Upload Product Images using Multer
Manage Product Details
Set Prices and Discounts
Role-Based Access Control
Security Features
JWT Authentication
Password Hashing with Bcrypt
Protected Routes
Session Management
Cookie-Based Authentication
