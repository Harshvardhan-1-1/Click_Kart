🛒 ClickKart

A full-stack e-commerce web application built using Node.js, Express.js, MongoDB, and EJS. ClickKart provides a seamless online shopping experience with secure user authentication, product management, image uploads, and shopping cart functionality.

## ✨ Features

### 🔐 Authentication & Security
- Secure user registration and login system
- JWT-based authentication
- Password hashing using Bcrypt
- Cookie-based session management
- Protected routes for authorized users only

### 🛍️ Product Management
- Create and manage products through the admin panel
- Upload product images using Multer
- Configure product pricing and discounts
- Dynamic product listing from MongoDB
- Product customization options

### 🛒 Shopping Cart
- Add products to cart
- View and manage cart items
- Persistent cart data for logged-in users
- Smooth shopping experience

### 👤 User Experience
- Dynamic server-side rendering with EJS
- Responsive and user-friendly interface
- Flash messages for real-time feedback
- Easy navigation across the application

### ⚡ Backend Functionality
- RESTful routing architecture
- MongoDB integration using Mongoose
- Environment variable management with Dotenv
- Express Session support
- Modular and scalable project structure

### 🏗️ Admin Controls
- Separate admin/owner routes
- Product creation and management
- Restricted access to administrative actions
- Inventory management workflow
